{\footnotesize\ttfamily \#define SHARED}



Useful to choose how to use the lib on Windows systems. 

If you want to use the lib with the dll, you don\textquotesingle{}t need to add anything in the command line. If you want to use the static version of the lib, then put \char`\"{}-\/\+D STATIC\char`\"{} in your command line when compiling, so you let the compiler know that the keyword \char`\"{}\+SHARED\char`\"{} is set to nothing and the function declarations are not provided with the \+\_\+\+\_\+declspec() attribute.
